A retail company has run multiple marketing campaigns and recorded detailed information about customers, their demographics, spending across product categories, channel-wise purchases, and whether they accepted specific campaigns.

This project has consolidated analytics solution to nderstand their best customers, caimpaign acceptance, and how different customer segments behave across products and channels.

The python notebook performs all the data importing, data celaning (handling missing values, extreme values, outliers and other data types), and data visualisation to understand customer behaviour according to various segments and analytics.

SQL Commands perform and depict the statistics of the business progress and top contributors.

The final python file has the code to connect the SQL database of the market dataset to streamlit app for users (stakeholders) to understand the dataset through visualisation and summaries, and also key insights and patterns along with recommendation to improve business.
